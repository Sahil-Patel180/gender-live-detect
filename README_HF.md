# Gender Classifier

AI-powered gender classification with privacy-first online learning.

## Features
- Real-time gender prediction from photos
- Privacy-safe learning (no image storage)
- Online learning from user feedback
- MobileNetV2 transfer learning
- Instant model updates

## Technology
- TensorFlow 2.20.0
- Gradio 4.44.0
- MobileNetV2 architecture
